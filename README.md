# Automobile Sales Recession Analysis Dashboard

## Project Overview

This project analyzes historical automobile sales data to understand the impact of economic recessions on vehicle sales. The analysis focuses on identifying trends, economic indicators, and seasonal patterns affecting automobile sales.

The project includes exploratory data analysis (EDA), multiple data visualizations, and an interactive dashboard built using Plotly Dash.

---

## Objectives

- Analyze automobile sales trends across different years
- Study the effect of recession periods on vehicle sales
- Explore the relationship between economic indicators and automobile sales
- Identify seasonal sales patterns
- Build an interactive dashboard for dynamic data exploration

---

## Dataset

The dataset contains synthetic automobile sales data used for analytical purposes.  
It includes the following variables:

- Date
- Year
- Month
- Recession
- GDP
- Consumer Confidence
- Unemployment Rate
- Automobile Sales
- Vehicle Type
- Advertising Expenditure
- Competition
- Seasonality Weight

## Data Analysis

The project performs several visual analyses including:

- Yearly automobile sales trend
- Advertising expenditure vs automobile sales
- Vehicle type sales trends
- GDP comparison during recession and non-recession periods
- Seasonality impact on automobile sales
- Price vs automobile sales analysis
- Advertising distribution analysis
- Unemployment rate impact on sales

Additional analysis includes:

- Correlation heatmap of economic indicators
- Recession vs non-recession sales comparison
- Vehicle category sales comparison
- Monthly sales trend analysis
- GDP vs automobile sales relationship

---

## Dashboard

An interactive dashboard was developed using Plotly Dash to allow dynamic exploration of automobile sales data.

The dashboard includes:

- Recession Report
- Yearly Report
- Vehicle type sales visualization
- Advertising vs sales analysis

Users can interact with dropdown filters to explore data across different years and report types.
