# Automobile-sales-recession-analysis
Automobile sales recession analysis using Python, data visualization, and interactive dashboard.
## Project Overview
This project analyzes historical automobile sales data to understand the impact of economic recessions on vehicle sales.

## Objectives
- Study automobile sales trends during recession periods
- Analyze the relationship between economic indicators and sales
- Visualize seasonal sales patterns
- Build an interactive dashboard

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Dash

## Key Insights
- Automobile sales decline during recession periods.
- Higher unemployment rates negatively affect automobile purchases.
- GDP and consumer confidence positively influence sales.
- Smaller family vehicles maintain stable sales during economic downturns.

## Dashboard
An interactive dashboard was created using Plotly Dash to explore automobile sales trends and economic indicators.

## Dataset
The dataset contains synthetic automobile sales data for analysis purposes.
